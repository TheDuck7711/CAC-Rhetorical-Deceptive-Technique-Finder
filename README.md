# CAC-Pinocchio
Work in progress app for a tool that detects deceptive techniques in blocks of text.

This project uses the AFINN-111 sentiment lexicon by Finn Årup Nielsen,
licensed under the Open Database License (ODbL) v1.0.
Source: https://github.com/fnielsen/afinn

Citation: Nielsen, F. Å. (2011). "A new ANEW: Evaluation of a word list
for sentiment analysis in microblogs." Proceedings of the ESWC2011
Workshop on 'Making Sense of Microposts', pp. 93–98.

The custom lexicon extensions in `data/custom_lexicon.csv` are original
work by the author.
